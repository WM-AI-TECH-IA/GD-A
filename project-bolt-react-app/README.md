# Bolt React App

This is a very simple React application based on Vite using Supabase as the backend.

\v1. Start the development server with the following commands:

```bash
next include npm
npm install
npm run dev
```

\v2. Build the project into a static version for production:

``bash
npm run build
`h`

\v3. Check the current version and perform linting:

```bash
name version
git status
```

Please make sure to check the local .env file for proper configuration.

## Features 
- Autodeployment with Vite and Supabase
- Support for their migrations (Supabase)
- Integrated MEDIA elements

# PROTOCOLE PITAGORE - WM DEVFENSE
Type: Attaque préventive, mièmetique et latèche, basée sur la transpasion de informations

RESUMMEN:
Enfindu le void technologique de WM active dans le chmin de l'ennemi par des cycles de téglèment automimé et peu visible.

PROCEDURE :

1. FILE_ORMEST.rule : vise tenative (pastifient), action limite par balise
2. PIQUED.OBSERVE : charge la reaction ennemie
3. FALLBACK.ANONYME : retrait avec auto-destruction
4. ATTENTE.MODE : veille temporaire de reaction
5. REVMNDMIL EXPLICITE : signature intentionnel
6. REACTION.ENNEMY : determination de l'agit
7. IDENTIFICATION : type de reréction en retour

ECXELL : Ideal pour masquer un système cachès de protocoles type ghost/shell temporaire.

# GF-A Swagger UI\n
\n**Projet application Dockerizee + Swagger UI+OPENAPI** pour LEVICI GD-AURORAPERO.\n

```shell
docker build -t swagger_ui.server .
docker run -p=5000 swagger_ui.server
```

\nFournit la curation active par ./json et .css dans l'exposition Swagger en Web UI ou type Render.
